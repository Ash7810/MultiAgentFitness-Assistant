# Multi-Agent Fitness Assistant

A multi-agent orchestration system for personalized health 
and fitness guidance using RAG and LLM APIs.

## Architecture
- **Supervisor Agent** — routes user intent to the right sub-agent
- **RAG Agent** — retrieves context from ChromaDB vector store
- **Dietary Compliance Agent** — allergen-safe meal recommendations

## Stack
`Python` `ChromaDB` `RAG` `Multi-Agent` `Gemini API` `FastAPI`
